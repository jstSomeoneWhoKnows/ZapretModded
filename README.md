# 🚀 Zapret-Modded

[![GitHub release (latest by date)](https://img.shields.io/github/v/release/jstSomeoneWhoKnows/ZapretModded)](https://github.com/jstSomeoneWhoKnows/ZapretModded/releases)
[![License](https://img.shields.io/badge/license-GPLv3-blue.svg)](LICENSE)
[![VirusTotal](https://img.shields.io/badge/VirusTotal-Checked-brightgreen)](https://www.virustotal.com)

> **Форк оригинального репозитория [Zapret](https://github.com/bol-van/zapret) от flowseal**  
> Все авторские права на оригинальный код сохранены.

Тот же мощный **Zapret**, но:
- 🇷🇺 с полной русской локализацией
- ⚡ ускоренным кодом
- 🧩 максимально простым и понятным интерфейсом
- 📁 чёткой структурой папок для удобного редактирования

**Главная цель** — помочь людям, не знакомым с программированием, легко обходить блокировки.

> 🛠 Проект находится в активной разработке. Любая обратная связь и отзывы очень приветствуются!

## 📦 Что нового в Version 2 (RELEASE4)

### 🌍 Глобальные изменения
- Полная оптимизация скриптов для более стабильной работы
- Улучшена совместимость с современными сетевыми протоколами

### 🎯 Улучшения стратегий
- Актуализированы списки доменов для обхода блокировок
- Повышена стабильность работы стратегий для **Roblox**, **YouTube** и **Discord**

### 🔧 Автозамена файла `hosts`
> [!WARNING]
> **Важно.** Автозамена может работать нестабильно при включённом антивирусе **Kaspersky**.  
> При ошибках используйте **ручную** установку файла `hosts`.  
> Если авторежим крайне необходим, **приостановите защиту Kaspersky не более чем на 1 минуту** (на ваш страх и риск — это может временно снизить безопасность системы).

### 🐛 Исправление багов
- Устранены критические ошибки, вызывавшие закрытие консоли в предыдущих версиях

### ⚡ Оптимизация
- Продолжена работа по ускорению инициализации и снижению потребления ресурсов

## ⚡ Быстрый старт

1. Скачайте последний релиз на странице **[Releases](https://github.com/jstSomeoneWhoKnows/ZapretModded/releases)**
2. Откройте файл-установщик 
3. Запустите `ZM_Manager.exe`
4. Выберите и запустите нужную стратегию
