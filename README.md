# ZAPRET-MODDED

Форк оригинального репозитория **Zapret** от *flowseal* (Все оригинальные авторские права сохраняются за репозиторием `zapret-discord-youtube`).

Тот же **zapret**, но с русской локализацией, ускоренным кодом и структурированный по папкам для удобства редактирования.

>  Проект все еще в работе, и отзывы очень сильно приветствуются!

---

## Что нового | Version 1.2.0

### 1. Новая стратегия для Roblox
* Папки-`lists` вынесены отдельно для удобства редактирования.
* Домены **Roblox** добавлены в проверку стратегий.

### 2. Обновлен `list-general`
* Комментариями выделен главный домен/сайт.

### 3. Автозамена файла `hosts`
* *Примечание:* При условии приостановленного антивируса на 1 минуту (из-за самозащиты блокируется доступ и замена файлов в папке `system32`).
* В случае ошибок предусмотрен стабильный ручной режим.

### 4. Исправление багов
* Исправлен баг с вылетом консоли при проверке лучшей стратегии (**Меню 11**).

### 5. Оптимизация
* Слегка ускорен код благодаря замене цветов на **ASCII**.
