ZAPRET-MODDED
Форк оригинального репозитория Zapret от flowseal (все оригинальные авторские права сохраняются за репозиторием zapret-discord-youtube).

Тот же zapret, но с русской локализацией, ускоренным кодом, максимально простым и понятным интерфейсом, а также структурированным разделением по папкам для удобства редактирования.

Репозиторий создан с целью помочь людям, незнакомым с программированием, обходить блокировки
🛠️ Проект всё ещё в разработке, и обратная связь (отзывы) очень приветствуется!

📌 Что нового | Version 2 (RELEASE4)
1. Глобальные изменения
Полная оптимизация скриптов для более стабильной работы.

Улучшена совместимость с современными сетевыми протоколами.

2. Улучшения стратегий
Актуализированы списки доменов для обхода блокировок.

Повышена стабильность работы стратегий для Roblox, YouTube и Discord.

3. Автозамена файла hosts
Автозамена может работать нестабильно при включенном антивирусе Kaspersky.

При возникновении ошибок рекомендуется воспользоваться ручным методом установки файла hosts.

Если вам обязательно нужен авторежим, можно приостановить защиту Kaspersky НЕ БОЛЕЕ ЧЕМ НА 1 МИНУТУ (делается на ваш страх и риск, так как это может подвергнуть устройство угрозам).

4. Исправление багов
Исправлены критические ошибки, вызывавшие закрытие консоли в предыдущих версиях.

5. Оптимизация
Продолжена работа по оптимизации кода для повышения скорости инициализации.
